# Quotes App

A simple motivational quotes Android application built using Kotlin in Android Studio.

## Features
- Display motivational quotes
- Random quote generator
- Share quotes with friends
- Simple and clean UI

## Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose

## How It Works
Click the "Next Quote" button to generate a new motivational quote.  
You can also share the quote using the "Share Quote" button.
